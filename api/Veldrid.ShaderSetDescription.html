Parameters
Type Name Description
VertexLayoutDescription[] vertexLayouts

An array of VertexLayoutDescription describing the set of vertex layouts understood by the Pipeline. Each element in this array describes the input layout of a single VertexBuffer to be bound when drawing.
ShaderStageDescription[] shaderStages

An array of ShaderStageDescription, one for each shader stage which is to be active in the Pipeline. At a minimum, every ShaderSetDescription must specify a Vertex and Fragment shader. All other stages are optional, but if either Tessellation stage is present, then the other must also be.
